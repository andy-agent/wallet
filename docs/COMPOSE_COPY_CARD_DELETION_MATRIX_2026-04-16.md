# Compose Copy-Card Deletion Matrix

Bead: `liaojiang-xn6o.6`

目标：为全 APP 批量删除说明型文案卡片提供单一实施矩阵。原则是先清运行时可达文案，再清页面层和 contract 默认值，最后清 preview/dev 残留。

## Card Categories

| Category | Meaning | Default Action |
| --- | --- | --- |
| `badge` | Hero/顶部状态徽标，如 `uiState.badge` | `delete` |
| `hero supportingText` | Hero 卡片副文案/拼接说明 | `delete` |
| `hero stats` | Hero 下方小卡片、metric pills、stats 列表 | `delete` |
| `note/summary` | 页面 summary、note、补充说明、说明卡 | `delete` |
| `highlights/checklist` | 说明型列表、状态检查、关键信息列表 | `delete` |
| `FeaturePageTemplate sections` | `输入信息 / 关键信息 / 状态说明 / 补充说明` 区块标题和区块本体 | `replace` with conditional rendering, then `delete` when empty |

## Batch Plan

按已确认顺序执行：

1. Shared components/runtime
   - 先改 [FeaturePageTemplate.kt](/Users/cnyirui/git/projects/liaojiang/code/Android/V2rayNG/app/src/main/java/com/v2ray/ang/composeui/components/feature/FeaturePageTemplate.kt)、[RealP0Repository.kt](/Users/cnyirui/git/projects/liaojiang/code/Android/V2rayNG/app/src/main/java/com/v2ray/ang/composeui/p0/repository/RealP0Repository.kt)、[RealCryptoVpnRepository.kt](/Users/cnyirui/git/projects/liaojiang/code/Android/V2rayNG/app/src/main/java/com/v2ray/ang/composeui/common/repository/RealCryptoVpnRepository.kt)、[P0Models.kt](/Users/cnyirui/git/projects/liaojiang/code/Android/V2rayNG/app/src/main/java/com/v2ray/ang/composeui/p0/model/P0Models.kt)

2. Mounted pages
   - 再清当前能点到的 mounted 页面：P0、P1、P2 core、P2 Extended mounted
   - 目标：移除 badge、hero supportingText、hero stats、note/summary、highlights/checklist

3. Contract defaults
   - 再清各 `Contract.kt` 默认值和 `FeatureField.supportingText`
   - 目标：contract 仅保留空值、真实值或必要输入标签，不再带结构性文案

4. Preview/dev residue
   - 最后清 `previewState`、`@Preview`、gallery 和 dev-only 残留
   - 目标：开发态彻底与运行态隔离，避免回流到可达 UI
