# Task Continuation Protocol\n\n## Recovery Commands\n1. bd ready --json\n2. bd list --status in_progress --json\n3. git status --short\n4. bash scripts/auto_resume_unfinished.sh --no-dispatch\n\n## First Output Contract\n- active critical-path tasks\n- dirty/untracked files\n- last verified checkpoint\n- next milestone\n\n## Auto Resume Default\n- Auto resume writes machine state first\n- Auto resume does not create placeholder AUTO tasks by default\n- Auto resume does not refresh human-facing status docs unless explicitly requested\n\n## Milestone Update Rules\n- Use bash scripts/auto_resume_unfinished.sh --milestone-docs when a milestone ends and human docs must be refreshed\n- Update docs/current-status.md\n- Update handoff/progress.md\n- Append docs/development-log.md\n- Update handoff/task-state.md\n\n## Interruption Handling\n- Keep milestone updates minimal and reversible\n- Persist next commands in handoff/progress.md\n\n## Rollback Notes\n- Revert with git restore <file> or git checkout -- <file>\n
