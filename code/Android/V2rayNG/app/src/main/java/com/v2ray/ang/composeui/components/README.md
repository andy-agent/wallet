# Components

This directory contains Compose UI components for the VPN app.

Components are organized by category:
- buttons/ - Button components
- cards/ - Card components
- inputs/ - Input field components
- tags/ - Status tag components
- dialogs/ - Dialog components
- listitems/ - List item components
- navigation/ - Navigation components
- special/ - Special purpose components

Note: This is a skeleton structure for 4j0.10 migration.
Full component implementations will be added in subsequent tasks.
